# Gadgets4Life Website

## Project Overview
Gadgets4Life is a simple yet functional website designed to showcase smartphones and electronic gadgets. The project was developed using core web technologies—HTML, CSS, and JavaScript—and focuses on front-end design, usability, and basic interactivity. It was created as part of an academic web development exercise.

## Objectives
- Apply HTML for structured and semantic web content  
- Use CSS to implement layout design, styling, and responsiveness  
- Use JavaScript to enhance user interaction and dynamic behaviour  
- Demonstrate understanding of UI design and client-side scripting  

## Technologies Used
- **HTML5** – Page structure and content  
- **CSS3** – Styling, layout, navigation, and responsive design  
- **JavaScript** – Interactive elements and basic functionality  

## Website Features
- Home page introducing the Gadgets4Life brand  
- Smartphone and gadget listings with images and descriptions  
- Navigation menu for easy access to different sections  
- Interactive UI elements such as buttons and hover effects  
- Responsive layout adapting to different screen sizes  
- Clean and consistent visual design

1. Download or clone the repository  
2. Open `index.html` in any modern web browser  
3. No additional installation or server setup is required
